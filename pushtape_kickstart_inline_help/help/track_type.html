Help on content types

content variations are defined here from within the “content types”. For each kind of content variation you can manage fields like categories, images and just about anything that is a field for Drupal.

The _content display nodes_ link these content types so you can add them inline. Generally, in Pushtape Kickstart, contents are created from the add content screen.
